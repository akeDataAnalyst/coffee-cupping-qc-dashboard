# Specialty Coffee Cupping QC Dashboard
[![Live Demo](https://img.shields.io/badge/Streamlit-Live%20Demo-brightgreen)](https://coffee-cupping-qc-dashboard-yhzkxu67ouklkkm2wtcr6r.streamlit.app/)

**A Modern Sensory Evaluation & Quality Control System for Ethiopian Specialty Coffee**

---

## Project Description

This is a professional Coffee Cupping QC Dashboard designed to simulate a modern coffee laboratory system. 

It allows users to record, analyze, visualize, and report sensory evaluation results using SCA (Specialty Coffee Association) protocols
---

## Problem Statement

- Cupping results in many labs and sourcing teams are recorded manually in spreadsheets.
- Lack of centralized, consistent, and visual data makes it difficult to compare lots, track trends, and generate professional client reports.
- Slow access to quality insights delays sourcing decisions and supplier feedback.
- Difficulty in demonstrating clear quality performance to international buyers.

---

## Solution

A clean, interactive Cupping Quality Control Dashboard that standardizes sensory data collection and provides powerful insights for decision-making.

### Key Features

- Record and store SCA cupping scores (10 attributes + total score)
- Automatic quality tier classification (Good → Outstanding)
- Filter and analyze by region, processing method, supplier, and score
- Interactive visualizations (score distribution, regional performance, attribute analysis)
- Professional data export (CSV) for client reports
- Alignment with ECX physical standards and SCA sensory protocols

---

## Tech Stack

- **Language**: Python 3.9+
- **Core Libraries**: Pandas, NumPy
- **Visualization**: Plotly Express
- **Dashboard Framework**: Streamlit
- **Data Validation**: Pydantic
- **Project Structure**: Modular `src/` package architecture
- **Data Formats**: CSV, Pickle, Excel
